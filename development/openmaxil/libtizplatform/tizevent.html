OMX_ERRORTYPE tiz_event_loop_init(void)¶

Explicit initialisation of the global event loop. The loop is hosted in its own thread which is spawned the first time this function or any other function in this module are called. Therefore it is not mandatory to call this function in order to instantiate the global event loop. This is only useful if for some reason the initialization cannot be done at the same time as the first use.

Return
OMX_ErrorNone if success, OMX_ErrorInsufficientResources otherwise.

void tiz_event_loop_destroy(void)¶

Explicit destruction of the global event loop. WARNING: After this function, the event loop cannot be recreated in the current process. Therefore, if this function is used, the caller should guarantee that the global event loop is no longer needed.
